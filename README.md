# 🌍 World Happiness Report Analysis (2015-2025)

## 📌 Deskripsi
Analisis data World Happiness Report untuk menemukan faktor-faktor 
yang mempengaruhi kebahagiaan suatu negara dan tren kebahagiaan global 
dari 2015 hingga 2025.

## 🎯 Pertanyaan Analisis
- Negara mana yang paling bahagia dan paling tidak bahagia?
- Bagaimana tren kebahagiaan global selama 10 tahun terakhir?
- Faktor apa yang paling mempengaruhi kebahagiaan?

## 📊 Visualisasi
- Top 10 Happiest Countries
- Global Happiness Trend (2015-2025)
- Correlation Heatmap
- GDP vs Happiness Scatter Plot
- Bottom 10 Least Happy Countries

## 🛠️ Tools & Library
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## 📁 Dataset
[World Happiness Report 2005-2025](https://www.kaggle.com/datasets/elvisbui/world-happiness-report-2005-2025-panel)

## 💡 Key Insights
- Finland konsisten menjadi negara paling bahagia
- Social Support adalah faktor terkuat kebahagiaan (korelasi 0.71)
- 8 dari 10 negara paling tidak bahagia berada di Afrika
- Kebahagiaan global meningkat signifikan di 2025 pasca pandemi
